## üá∫üá∏ English

**Discord Quest Automator** is a lightweight, GUI-based tool designed to automatically complete Discord Quests (Stream/Play tasks) without actually installing games or streaming. It works by injecting a script into the Discord client to emulate game activity.

### ‚ú® Features
* **Modern UI:** Clean, dark-themed interface built with PyQt6.
* **Real-time Tracking:** Live progress bar and timer synced with Discord.
* **Auto-Detection:** Automatically finds active quests and required tasks.
* **Background Work:** Runs silently; no need to keep Discord focused.
* **Multi-Task:** Can handle video, gaming, and streaming tasks.

### üöÄ Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/Discord-Quest-Automator.git](https://github.com/YOUR_USERNAME/Discord-Quest-Automator.git)
    cd Discord-Quest-Automator
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the application:**
    ```bash
    python main.py
    ```
